# 露营装备（新手）

## 定义

面向**首次或少量经验**露营者的装备集合与优先级：通常以**睡眠系统**（帐篷、睡袋、睡垫）为核心，其次为照明、炊事与安全用品；根据场景分为近郊 car camping、国家公园营地与多日徒步/飞行携带等变体。

## 装备清单

### 睡眠系统（核心三件套——决定体验上限）

| 装备 | 选购要点 | 参考品牌 |
|------|----------|----------|
| **帐篷** | 按人数 ×2 选容量；新手选快开/instant 款；配**地布**铺帐篷下方 | Coleman Instant Tent |
| **睡袋** | 按营地最低温度选温标；信封式热了可打开当被子 | — |
| **睡垫** | 支撑度直接影响睡眠；别买便宜薄充气床垫 | Exped（贵但好）、Losthorizon（平替）、Sea to Summit |

### 安全与照明

| 装备 | 说明 |
|------|------|
| **头灯** | ≥300 流明；夜间行走、上厕所必备 |
| **营地灯** | 挂帐篷内；注意与帐篷挂钩兼容；灯宜多带 |
| **急救包** | 绷带、消毒湿巾、镊子、止痛药 |
| **移动电源** | 短途普通充电宝即可；长期 camping 可考虑大容量户外电源 |

### 炊事

| 装备 | 说明 |
|------|------|
| **小气炉 + 气罐** | 美国超市气罐便宜，不必户外店买；注意规格匹配 |
| 气罐用量参考 | 8oz 约烧 1–1.5h，够双人一顿晚饭 + 简单早餐 |
| **锅/烤盘** | 可带家里的；烤盘**勿盖住气罐上方**（受热危险） |

### 其他

| 装备 | 说明 |
|------|------|
| 露营椅 + 桌 | 沃尔玛便宜好用；部分营地自带野餐桌 |
| 杀虫剂 / 防晒 | 水边营地蚊虫多，帐篷也喷 |
| 垃圾袋 | Leave No Trace |
| 多功能工具 | 刀具 + 锤子（打地钉） |
| 充气枕头 | 非必须但加分 |
| 滤水设备 | 看营地取水条件 |

### Solo 飞美西 + Hiking 额外项

衣物分层（城市与山里通用）、轻量化优先、**熊罐**（部分区域强制）。详细清单在图片中，见 [[小红书｜一人美西露营与 Hiking 装备清单]] 原帖截图。

## 新手策略（多篇共识）

1. **能借则借、能用家用则不买满**——第一次发现不喜欢，装备不至于闲置。
2. **出发前在家试搭帐篷、试充睡垫**——到了营地天黑手忙脚乱。
3. **炊事可以晚一步**——集体露营蹭大神、自己露营第一次打包外卖也行。
4. **衣物多穿几层**，随气温增减；别忘**拖鞋**。
5. **睡眠投入 > 其他一切**——睡好了露营就成功一半。
